# Playback Cifras IA v2.9

Objetivo:
Corrigir a biblioteca do Google Drive para montar músicas reais a partir dos arquivos da pasta selecionada.

Estrutura esperada:
ROCK/
- So Far Away.pdf
- So Far Away.mp3

Implementação:
- Agrupar arquivos por nome base.
- Associar PDF e MP3 da mesma música.
- Ignorar pc_songs quando Google Drive estiver conectado.
- Popular pdfFileId e audioFileId.
- Abrir PDF e tocar MP3 ao selecionar a música.
