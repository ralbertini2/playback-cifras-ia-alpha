# Playback Cifras IA v2.3 — React Player

## Branch

```text
feature/v2-3-react-player
```

## Objetivo

Criar o novo Player React do Playback Cifras IA como módulo independente da nova arquitetura.

## Inclui

- PlayerBar.
- PlaybackControls.
- ProgressBar.
- VolumeControl.
- Hook `useAudioPlayer` evoluído.
- Service `audioService.js`.
- Play/Pause.
- Barra de progresso com seek.
- Tempo atual e duração total.
- Música anterior e próxima.
- Avançar e voltar 10 segundos.
- Controle de volume.
- Mute.
- Layout responsivo Tablet First.

## Não inclui

- Google Drive final.
- Repertórios finais.
- Favoritos.
- IA.
- Sincronização PDF ↔ áudio.

## Validação esperada

```bash
npm install
npm run build
npm run preview
```
