---

## Objetivo

Simplificar a interface principal removendo os botões **Modo Palco** e **Tela Cheia**, além dos códigos diretamente associados a esses recursos.

Esta alteração reduz complexidade visual e prepara a base para uma experiência Tablet/iPad Vertical mais limpa.

---

## Alterações Implementadas

### Removed

- Botão `Modo palco` removido da barra superior.
- Botão `Tela cheia` removido da barra superior.
- Eventos de clique relacionados a `stageBtn` e `fullscreenBtn`.
- Estado persistido `pc_stage_mode` da configuração `STORAGE`.
- Funções `toggleStage`, `updateStageBtn` e `fullscreen`.
- Atalho de teclado `M` para alternar modo palco.
- Regras CSS específicas de `body.stage`.
- Regras responsivas específicas dos botões removidos.

### Changed

- Barra superior permanece apenas com controles úteis para leitura, zoom, rolagem e favoritos.
- Interface fica mais limpa para evolução Tablet First.
