Branch:

---

# Objetivo

Iniciar a integração real do Google Drive na arquitetura React/Vite, conectando a biblioteca do Drive ao visualizador PDF e ao player React.

---

# Alterações Implementadas

## Added

- Hook `useGoogleDriveLibrary`.
- Estado React para conexão Google Drive.
- Estado React para biblioteca de músicas.
- Estado React para pasta raiz do Drive.
- Estado React para música atual.
- Login/logout Google.
- Seleção de pasta com Google Picker.
- Atualização da biblioteca via Google Drive API.
- Carregamento autorizado de PDF.
- Carregamento autorizado de áudio.
- Estados de carregamento para biblioteca e seleção de música.

## Changed

- `App.jsx` passa a usar `useGoogleDriveLibrary`.
- `Sidebar` recebe estado de carregamento.
- `Toolbar` exibe indicador de carregamento.
- Versão atualizada para `v2.4`.
