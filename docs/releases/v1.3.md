# Objetivo

Reduzir a complexidade da base v1 removendo telas, serviços e estruturas que ainda não fazem parte do escopo funcional atual.

A prioridade desta versão é manter o sistema mais estável para evoluções futuras, preservando a integração com Google Drive, PDFs, playbacks, favoritos e playlists/eventos.
