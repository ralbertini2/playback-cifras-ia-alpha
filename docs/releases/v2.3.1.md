# Playback Cifras IA v2.3.1 — React Runtime Validation

## Branch

```text
feature/v2-3-1-react-runtime-validation
```

## Objetivo

Validar que a fundação React/Vite do Playback Cifras IA consegue instalar dependências e gerar build sem erros antes da integração com Google Drive.

## Alterações

- Adicionado workflow `React Build Validation`.
- Workflow executa `npm install` e `npm run build`.
- Adicionado script `validate` no `package.json`.
- Atualizada versão do projeto para `2.3.1`.
- Atualizado `APP_VERSION` para `v2.3.1`.
- Adicionado `public/config.js` seguro para evitar erro de configuração ausente no runtime inicial.
- Adicionado `.gitignore` para `node_modules`, `dist` e arquivos locais.

## Fora do escopo

- Google Drive final.
- Player conectado a arquivos reais.
- PDF conectado ao Drive.
- Repertórios.
- Favoritos.

## Critérios de aprovação

- GitHub Actions executa sem erro.
- `npm install` conclui com sucesso.
- `npm run build` conclui com sucesso.
- Nenhuma alteração funcional crítica é introduzida.
