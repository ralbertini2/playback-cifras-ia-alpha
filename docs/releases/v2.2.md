# Playback Cifras IA v2.2 — React PDF/Cifra Viewer

## Branch

```text
feature/v2-2-pdf-viewer
```

## Objetivo

Criar o primeiro módulo funcional da arquitetura React: o visualizador de PDF/cifras.

## Implementado

- PDF Viewer em React.
- Integração inicial com PDF.js.
- Hook `usePdfViewer`.
- Service `pdfService.js`.
- Controles de zoom.
- Ajuste à largura.
- Página anterior/próxima página.
- Indicador de página.
- PDF de exemplo para validação.
- Layout responsivo Tablet First.

## Fora do escopo

- Integração final com Google Drive.
- Player final.
- Repertórios.
- Favoritos.
- IA.

## Validação esperada

```bash
npm install
npm run build
npm run preview
```
