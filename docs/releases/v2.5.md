# Playback Cifras IA v2.5 — React Library

## Branch

```text
feature/v2-5-library-setlists-react
```

## Objetivo

Implementar a Biblioteca React do Playback Cifras IA, organizando músicas vindas do Google Drive e preparando a base para repertórios mais avançados.

## Inclui

- Componente `Library`.
- Componente `Setlists`.
- Hook `useLibrary`.
- Service `libraryService.js`.
- Busca por música, artista ou estilo.
- Filtros: todas, favoritas e recentes.
- Favoritos persistidos em `localStorage`.
- Recentes persistidos em `localStorage`.
- Integração da biblioteca com Sidebar.
- Integração do favorito com Toolbar.
- Versão atualizada para `v2.5`.

## Não inclui

- Repertórios avançados com drag and drop.
- Sincronização de repertórios na nuvem.
- IA.
- Publicação da v2 em produção.

## Validação esperada

```bash
npm install
npm run build
```

O workflow `React Build Validation` deve permanecer verde.
