# Playback Cifras IA v3.0.0 — Drive PDF Fix

## Objetivo
Corrigir a abertura de PDFs carregados a partir do Google Drive, mantendo o MP3 funcional após a v2.9.

## Causa raiz
A biblioteca do Google Drive já encontrava `pdfFileId` e `audioFileId`, e o MP3 já era convertido em URL autorizada corretamente. O erro estava no envio da fonte para o PDF.js: a versão em uso espera uma fonte explícita no formato `{ url: source }`, mas o serviço enviava a string diretamente para `getDocument()`.

## Correções
- `src/services/pdfService.js`: normaliza fontes string para `{ url: source }` antes de chamar `pdfjs.getDocument()`.
- `src/hooks/useGoogleDrive.js`: limpa URLs de mídia ao desconectar o Google Drive, remover pasta ou carregar biblioteca vazia.
- `package.json`: remove dependências `latest`, fixa versões estáveis e move Vite/plugin React para `devDependencies`.
- `package-lock.json`: incluído para builds reproduzíveis no GitHub Actions.
- Atualização de versão para `v3.0.0`.

## Validação
- `npm install` executado.
- `npm run build` executado com sucesso.
- Mantido fluxo Google Drive → Biblioteca → PDF → MP3.
